Dochris жҳҜдёҖдёӘ AI й©ұеҠЁзҡ„зҹҘиҜҶеә“зј–иҜ‘зі»з»ҹпјҢйҖҡиҝҮ**еӣӣйҳ¶ж®өжөҒж°ҙзәҝ**е’Ң**еӣӣеұӮдҝЎд»»жЁЎеһӢ**пјҢе°Ҷж•ЈиҗҪзҡ„ж–Үд»¶иҪ¬еҢ–дёәз»“жһ„еҢ–гҖҒеҸҜжҗңзҙўгҖҒеҸҜйӘҢиҜҒзҡ„зҹҘиҜҶеә“пјҢе№¶ж”ҜжҢҒдёҺ Obsidian еҸҢеҗ‘еҗҢжӯҘгҖӮ

## дёәд»Җд№ҲйҖүжӢ© Dochrisпјҹ

- **з»“жһ„еҢ–иҫ“еҮә**пјҡиҮӘеҠЁжҸҗеҸ–ж‘ҳиҰҒгҖҒе…ій”®зӮ№гҖҒжҰӮеҝөпјҢдёҚеҶҚжҳҜйӣ¶ж•Јзҡ„ж–Үд»¶
- **иҙЁйҮҸжҠҠе…і**пјҡеӨҡз»ҙеәҰиҜ„еҲҶзі»з»ҹпјҲ0-100 еҲҶпјүпјҢзЎ®дҝқеҸӘжңүй«ҳиҙЁйҮҸеҶ…е®№иҝӣе…ҘзҹҘиҜҶеә“
- **дҝЎд»»еҲҶеұӮ**пјҡеӣӣеұӮдҝЎд»»жЁЎеһӢпјҢд»Һ LLM з”ҹжҲҗеҲ°дәәе·ҘзІҫйҖүпјҢйҖҗжӯҘжҸҗеҚҮеҸҜдҝЎеәҰ
- **еӨҡж јејҸж”ҜжҢҒ**пјҡPDFгҖҒйҹійў‘гҖҒи§Ҷйў‘гҖҒз”өеӯҗд№ҰгҖҒж–Үз« пјҢдёҖдёӘзі»з»ҹе…Ёжҗһе®ҡ
- **Obsidian иҒ”еҠЁ**пјҡж”ҜжҢҒеҸҢеҗ‘еҗҢжӯҘпјҢй«ҳиҙЁйҮҸеҶ…е®№еҸҜжҺЁйҖҒеӣһ Obsidian з¬”и®°еә“

## еҠҹиғҪзү№жҖ§

### еӣӣйҳ¶ж®өжөҒж°ҙзәҝ

| йҳ¶ж®ө | еҠҹиғҪ | иҫ“еҮә |
|------|------|------|
| **Phase 1 ж‘„е…Ҙ** | жү«жҸҸж–Үд»¶гҖҒеҺ»йҮҚгҖҒеҲӣе»ә manifest | `manifests/sources/SRC-NNNN.json` |
| **Phase 2 зј–иҜ‘** | LLM жҸҗеҸ–з»“жһ„еҢ–еҶ…е®№ | `outputs/` (ж‘ҳиҰҒ+жҰӮеҝө) |
| **Phase 3 е®Ўж ё** | иҙЁйҮҸиҜ„еҲҶгҖҒжұЎжҹ“жЈҖжөӢ | иҙЁйҮҸеҲҶж•° 0-100 |
| **Phase 4 еҲҶеҸ‘** | жҷӢеҚҮдҝЎд»»еұӮгҖҒеҗҢжӯҘ Obsidian | `wiki/` вҶ’ `curated/` вҶ’ Obsidian |

- OpenAI е…је®№зҡ„ API Key

### е®үиЈ…

```bash
# е…ӢйҡҶйЎ№зӣ®
git clone https://github.com/caozhangqing85-cyber/dochris.git
cd dochris

# еҲӣе»әиҷҡжӢҹзҺҜеўғ
python3.11 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# е®үиЈ…дҫқиө–
pip install -e .

# еҲӣе»әй…ҚзҪ®ж–Үд»¶
cp .env.example .env
# зј–иҫ‘ .env еЎ«е…ҘдҪ зҡ„ API Key
```

### й…ҚзҪ®

еңЁ `.env` ж–Үд»¶дёӯж·»еҠ пјҡ

```bash
# LLM API й…ҚзҪ®пјҲеҝ…йңҖпјү
OPENAI_API_KEY=your_api_key_here
OPENAI_API_BASE=https://open.bigmodel.cn/api/paas/v4
MODEL=glm-5.1

# е·ҘдҪңеҢәи·Ҝеҫ„пјҲеҸҜйҖүпјҢй»ҳи®Ө ~/.knowledge-baseпјү
WORKSPACE=~/.knowledge-base
```

### дҪҝз”ЁжөҒзЁӢ

```bash
# 1. жҹҘзңӢзі»з»ҹзҠ¶жҖҒ
kb status

# 2. ж‘„е…Ҙж–Үд»¶пјҲжү«жҸҸжәҗж–Үд»¶е№¶еҲӣе»ә manifestпјү
kb ingest

# 3. зј–иҜ‘пјҲLLM жҸҗеҸ–з»“жһ„еҢ–еҶ…е®№пјү
kb compile 10          # зј–иҜ‘еүҚ 10 дёӘж–Үд»¶
kb compile --concurrency 4  # дҪҝз”Ё 4 дёӘе№¶еҸ‘

# 4. жҹҘиҜўзҹҘиҜҶеә“
kb query "иҙ№жӣјжҠҖе·§"

# 5. иҙЁйҮҸжЈҖжҹҘдёҺжҷӢеҚҮ
kb quality --report
kb promote SRC-0001 --to wiki
kb promote SRC-0001 --to obsidian
```

### 5 еҲҶй’ҹеҝ«йҖҹдҪ“йӘҢ

```bash
# 1. еҲӣе»әжөӢиҜ•зӣ®еҪ•
mkdir -p ~/test-kb/raw/pdfs
cp your-file.pdf ~/test-kb/raw/pdfs/

# 2. и®ҫзҪ®е·ҘдҪңеҢә
export WORKSPACE=~/test-kb

# 3. ж‘„е…Ҙж–Үд»¶
kb ingest

# 4. зј–иҜ‘
kb compile

# 5. жҹҘзңӢз»“жһң
cat ~/test-kb/outputs/summaries/SRC-0001.md
```

A: иҝҷжҳҜжҷәи°ұ API зҡ„еҶ…е®№е®Ўж ёжңәеҲ¶гҖӮзі»з»ҹдјҡиҮӘеҠЁжё…жҙ—ж•Ҹж„ҹиҜҚпјҢеҰӮд»ҚйҒҮеҲ°й—®йўҳпјҢеҸҜе°қиҜ•дҪҝз”Ёе…¶д»– API жҸҗдҫӣе•ҶгҖӮ

### Q: иҙЁйҮҸиҜ„еҲҶжҖ»жҳҜ 10 еҲҶ

A: иҝҷжҳҜиҜ„еҲҶз®—жі•дёҺжЁЎеһӢиҫ“еҮәдёҚеҢ№й…Қзҡ„й—®йўҳгҖӮйҮҚиҜ•зј–иҜ‘йҖҡеёёдјҡиҺ·еҫ—жӯЈзЎ®еҲҶж•°гҖӮ

### Q: еҰӮдҪ•еҠ еҝ«зј–иҜ‘йҖҹеәҰ

A: дҝ®ж”№ `MAX_CONCURRENCY` еҸӮж•°жҸҗй«ҳе№¶еҸ‘ж•°пјҲй»ҳи®Ө 3пјүпјҢжҲ–дҪҝз”Ё `nohup` еҗҺеҸ°иҝҗиЎҢгҖӮ
