# 质量管理

dochris 内置质量评估系统，确保知识库内容质量。

## 质量评分

每个编译文档自动评分（0-100），评估维度：

| 维度 | 权重 | 说明 |
|------|------|------|
| 完整性 | 25% | 是否覆盖原文主要内容 |
| 准确性 | 25% | 概念提取是否准确 |
| 结构化 | 25% | 摘要是否清晰有条理 |
| 关联性 | 25% | 是否建立了有效的知识关联 |

## 质量门槛

默认阈值 85 分。低于阈值的文档：

```bash
# 查看低质量文档
kb quality

# 生成质量报告
kb quality --report

# 重新编译低质量文档
kb compile --quality 85
```

## 批量质量操作

```bash
# 批量晋升高质量文档
kb promote --all --quality 90
```

## 质量监控

持续监控知识库质量状态：

```bash
# 系统状态（包含质量统计）
kb status
```
